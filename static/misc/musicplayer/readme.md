# Monochrome

**Monochrome** is an open-source, privacy-respecting, ad-free [TIDAL](https://tidal.com) web UI, built on top of [Hi-Fi](https://github.com/sachinsenal0x64/hifi).

Check it out live at: [**monochrome.tf**](https://monochrome.tf)
